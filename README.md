# Site internet Groupe Solid — groupe-solide.com

Construction, rénovation et promotion immobilière à Koh Samui.

## Contenu
- `index.html` — le site complet (une page, FR/EN/TH intégrés)
- `sitemap.xml` / `robots.txt` — fichiers pour Google
- `images/` — photos de chantier et fiches terrain (à ajouter)
